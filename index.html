Syntax Additions

Arrow Functions

Arrow functions in ES2015 are reminiscent of lambda expressions in .NET, if only in form.

Old way:

[1,2,3,4].map(function(item) {
    return item * 2;
}); // [2,4,6,8]

With arrow functions:

[1,2,3,4].map(item => item * 2); // [2,4,6,8]

You can see how much more concise the code can be.

See arrows.js for more examples.

See arrows.html for more information.

`for .. of` Loops

While on the subject of iteration, a new type of of loop was added, `for .. of`, which iterates over an iterable object.

Old way 1 (using normal indexed `for` loop)

var array = [1,2,3,4];
for (var i = 0, z = array.length; i < z; i++) {
    console.log(array[i]);
}
/*
logs:
1
2
3
4
*/

Old way 2 (using protected `for .. in` loop)

var array = [1,2,3,4];
for (var i in array) {
    // the following filter is needed so that properties like `length` are not enumerated also
    if (array.hasOwnProperty(i)) {
        console.log(array[i]);
    }
}
/*
logs:
1
2
3
4
*/

New way

const array = [1,2,3,4];
for (let item of array) {
    console.log(item);
}
/*
logs:
1
2
3
4
*/

See for-of.js for more examples.

ES2016 aka ES7

Note that in 2016, ECMAScript TC39 changed its naming scheme to use the date, rather than a version number, to identify the release. They backdated the change to ES2015, but since the proverbial cat was out of the bag, it did not take.

Concepts and Types

No new types were added in 2016.

Methods and Properties of Built-in Types

Array

includes

Determines if the argument exists in the current array.

const array = [1,2,3,4];
const exists = array.includes(3);
const doesNotExist = array.includes(5);
console.log(exists);
console.log(doesNotExist);
/*
logs:
true
false
*/

Syntax Additions

Exponentiation

The ** operator allows arbitrary exponentiation of numbers, previously only possible with Math.pow(). 


console.log(2 ** 3);
console.log(2 ** 1.5);
/*
logs:
8
2.82842712474619
*/

Methods and Properties of Built-in Types

Object

values

Returns an array of the values of enumerable own properties.


const obj = {
  a: 1,
  b: 'bee',
  c: 3.14,
  d: [5,6,7]
};
const result = Object.values(obj);
console.log(result);
/*
logs:
[
  1,
  "bee",
  3.14,
  [5,6,7]
]
*/
entries

Returns an array of arrays, with each sub-array having a key and a value element, for each own property of the object. 


const obj = {
  a: 1,
  b: 'bee',
  c: 3.14,
  d: [5,6,7]
};
const result = Object.entries(obj);
console.log(result);
/*
logs:
[
  [ "a", 1 ],
  [ "b", "bee" ],
  [ "c", 3.14 ],
  [ "d", [5,6,7] ]
]
*/
getOwnPropertyDescriptors

Returns an object containing the own property descriptors of an object (see ToPropertyDescriptor for information on the shape of the object). 


const obj = {
  a: 1,
  b: 'bee',
  c: 3.14,
  d: [5,6,7]
};
const result = Object.getOwnPropertyDescriptors(obj);
console.log(result);
/*
logs:
{
  "a": {
    "value": 1,
    "writable": true,
    "enumerable": true,
    "configurable": true
  },
  "b": {
    "value": "bee",
    "writable": true,
    "enumerable": true,
    "configurable": true
  },
  "c": {
    "value": 3.14,
    "writable": true,
    "enumerable": true,
    "configurable": true
  },
  "d": {
    "value": [
      5,
      6,
      7
    ],
    "writable": true,
    "enumerable": true,
    "configurable": true
  }
}
*/
